# OptiConnect Solutions – Call Center Performance Dashboard

## Project Overview
This project analyzes call center performance data for OptiConnect Solutions using Power BI.
The dashboard evaluates operational efficiency, agent performance, and customer satisfaction.

## Key Objectives
- Analyze call answer and resolution performance
- Evaluate agent-wise and department-wise efficiency
- Study the impact of speed of answer on customer satisfaction
- Provide actionable insights for management

## Files in this Repository
- `PowerBI_Dashboard.pbix` – Interactive Power BI dashboard
- `Dataset/Call Centre Dataset.xlsx` – Raw dataset used for analysis
- `Dashboard_PDF/OptiConnect_Dashboard.pdf` – Static dashboard export
- `Screenshots/` – Preview images of the dashboard

## How to View the Dashboard
1. Download `PowerBI_Dashboard.pbix`
2. Open using Power BI Desktop
3. If prompted, set the dataset path to the Excel file inside the Dataset folder

## Tools Used
- Power BI Desktop
- Microsoft Excel
